How to enable DNS over HTTPS in Firefox

DNS over HTTPS reduces the amount of data your ISP can gather on you

  1. Click the menu button and select Options
  2. In the General panel, scroll down to Network Settings and click the Settings button.
  3. In the dialog box that opens, scroll down to Enable DNS over HTTPS.
  4. Select the Enable DNS over HTTPS checkbox. Select Cloudflare as your provider.
  5. Click OK to save your changes and close the window.

How to enable DNS over HTTPS in Chrome

  1. Open the menu or preferences page
  2. In the right hand menu select Privacy and security
  3. Scroll down to secure DNS
  4. Enable secure DNS
  5. Change the option from With you current provider to with Cloudflare (1.1.1.1)
